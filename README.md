# Excel Data Analysis Project 📊

> An end-to-end Excel data analysis project focused on cleaning, transforming, analyzing, and visualizing customer purchasing data through Pivot Tables and an interactive dashboard.

This project demonstrates how Microsoft Excel can be used as a powerful business intelligence tool to transform raw customer data into meaningful insights for decision-making.

---

# Project Overview

The project analyzes customer demographic data to identify factors that influence bike purchasing behavior.

The workflow follows a typical data analysis process:

1. Data Cleaning
2. Data Transformation
3. Data Analysis
4. Data Visualization
5. Dashboard Creation

---

# Dataset

The dataset contains customer information including:

- Customer ID
- Marital Status
- Gender
- Income
- Number of Children
- Education
- Occupation
- Home Ownership
- Number of Cars
- Commute Distance
- Region
- Age
- Bike Purchase Status

---

# Project Workflow

## 📥 Raw Data

The original customer dataset serves as the starting point for the analysis.

---

## 🧹 Data Cleaning

The data was prepared by:

- Checking for inconsistencies
- Standardizing values
- Formatting data
- Creating calculated fields where necessary
- Preparing the dataset for analysis

---

## 📈 Data Analysis

The cleaned data was analyzed using Pivot Tables to explore relationships between customer characteristics and purchasing decisions.

Key analytical dimensions include:

- Income vs Bike Purchases
- Age Distribution
- Commute Distance
- Gender Analysis
- Marital Status
- Regional Comparison

---

## 📊 Dashboard

An interactive dashboard was created to summarize key business insights.

The dashboard allows users to quickly identify trends and compare customer segments through dynamic charts and Pivot Table visualizations.
